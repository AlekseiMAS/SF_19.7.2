# SF_19.7.2

Тестирование REST API интерфейса сайта PetFriends.

Посмотреть документацию к имеющимся API-методам на сайте. 
Найти методы, которые ещё не реализованы в библиотеке, и написать их реализацию в файле api.py.

В файле settings.py расположенном в корневой директории содержится информация о валидных и невалидных емейлах и паролях.

В файле api.py расположенном в корневой директории содержится библиотека к REST api сервису веб приложения Pet Friends.

В папке /tests находится файл с тестами и фото для тестов.
